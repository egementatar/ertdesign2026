Supabase Kurulum Rehberi

1. supabase.com adresine gidin ve proje oluşturun.
2. Project Settings → API bölümünden Project URL ve anon public key değerlerini kopyalayın.
3. Aşağıya yapıştırın ve Bağlan butonuna tıklayın.

SQL Editor'da çalıştırmanız gereken tablo sorgusu:
CREATE TABLE messages (id uuid default gen_random_uuid() primary key, name text, email text, phone text, service text, message text, created_at timestamptz default now()); CREATE TABLE site_content (id uuid default gen_random_uuid() primary key, key text unique, value text, updated_at timestamptz default now()); CREATE TABLE gallery (id uuid default gen_random_uuid() primary key, title text, url text, "order" int, created_at timestamptz default now()); ALTER TABLE messages ENABLE ROW LEVEL SECURITY; CREATE POLICY "anon insert" ON messages FOR INSERT TO anon WITH CHECK (true); CREATE POLICY "auth read" ON messages FOR SELECT TO authenticated USING (true);
